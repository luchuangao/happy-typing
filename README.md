# 快乐打字（Happy Typing）

一款面向儿童与初学者的浏览器打字练习工具：大字体、清晰反馈、课程式键位训练，并支持自定义词库练习（词语/英文单词/短语等）。

## 功能亮点

- 课程训练：按「基准键位 / 上排 / 下排 / 括号 / 数字 / 符号」等课程逐步练习
- 自定义练习：输入你自己的词语或单词列表，系统随机抽取练习
- 无需输入框：点击页面空白处即可获得焦点，直接按键盘练习
- 友好反馈：正确计数、准确率显示；输错会自动撤销本次错误输入，无需删除
- 声音开关：支持音效与配音（可关闭，记忆偏好）
- 屏幕键盘：高亮下一个目标按键，辅助建立键位直觉

## 本地运行

由于使用了 ES Module（`type="module"`），建议用本地静态服务器访问。

```bash
python3 -m http.server 5173
```

浏览器打开：

http://localhost:5173/

## 使用说明

1. 选择模式：
   - 字母：随机 a-z
   - 英文：常见英文单词
   - 汉语拼音：不带声调，偶尔出现两个音节（中间空格）
   - 汉字：支持中文输入法，候选上屏后与目标一致即可通过
   - 课程训练：按课程逐步练习键位
   - 自定义：根据你输入的词语列表练习
2. 点击「开始」进入倒计时，然后直接按键盘输入目标内容
3. 输入错误会自动回退到上一次正确前缀，不需要手动按删除键

## 课程列表（课程训练）

- 01 基准键位：asdfjkl;
- 02 基准键位扩展：asdfghjkl;
- 03 基准键位与 Shift：ASDFGHJKL:
- 04 引号：' "
- 05 上排键（小写）：qwertyuiop
- 06 下排键（小写）：zxcvbnm,./
- 05 上排键（大写）：QWERTYUIOP
- 06 下排键（大写）：ZXCVBNM,./?
- 07 复习字母
- 08 括号：{}[]<>
- 09 数字键：1234567890
- 10 数字键与 Shift：!@#$%^&*()
- 11 数学符号键：- = _ +
- 12 复习全部

## 自定义词库（自定义模式）

在顶部「自定义词语」输入框中粘贴你的内容即可开始练习，支持分隔符：

- 空格
- 逗号（英文逗号或中文逗号）
- 换行

示例：

```
apple banana orange
hello, world, typing
北京 上海 广州
```
